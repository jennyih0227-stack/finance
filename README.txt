誠毅傳承 · 財經自學站　部署與使用說明
================================================

【這是什麼】
一個給財經小白的自學站（PWA 網站），含五個工具與五份教學簡報。
純靜態網頁，可直接放在 GitHub Pages 或你的個人網站。

------------------------------------------------
【檔案清單】

網頁（進入點）
  index.html      首頁 · 工具總覽
  cards.html      財經觀念小卡
  decode.html     新聞判讀工具
  read.html       教學簡報手機閱讀
  talk.html       客戶說明講稿速查
  report.html     每日財經速報

PWA 設定與圖示
  manifest.json   sw.js
  icon-192.png    icon-512.png    icon-maskable-512.png    apple-touch-icon.png

資料檔
  brief.json      每日速報的內容（每天更新這個檔）

教學簡報（首頁可下載 / 手機閱讀）
  財經新聞判讀術_教學簡報.pptx
  財經判讀_台股重返47K_教學簡報.pptx
  財經判讀_日企信心8年最高_教學簡報.pptx
  財經判讀_亞歐製造業PMI_教學簡報.pptx
  財經判讀_華許7月升息_教學簡報.pptx

客戶講稿（Word · 給你自己印/練，非網站必需）
  客戶說明講稿_財經新聞轉客戶對話.docx
  客戶說明講稿_放大版.docx      （字大、每則一頁滿版）

------------------------------------------------
【怎麼部署】

方式 A：GitHub Pages
  1. 把「網頁 / PWA / 資料檔 / 教學簡報」這些檔案放進 repo 的一個資料夾
     （例如 finance/），保持同一層、不要分子資料夾。
  2. 開啟 GitHub Pages，進入點即 index.html。
  3. 網址例：https://jennyih0227-stack.github.io/finance/
  ※ PWA 需要 HTTPS 才能「加到主畫面」；GitHub Pages 本身就是 HTTPS。

方式 B：個人網站
  把上述檔案上傳到網站的某個目錄即可（同一層）。
  若你的網站已經有自己的 manifest / service worker，
  兩套 PWA 設定可能衝突，需要時再合併 manifest.json / sw.js。

------------------------------------------------
【手機安裝（PWA）】
  Android Chrome：會自動跳出「加到主畫面」，或選單 →「安裝應用程式」。
  iPhone Safari：分享 → 加入主畫面。
  安裝後可離線開啟；長按圖示有捷徑直達各工具。

------------------------------------------------
【每天更新「每日財經速報」】
  1. 開 report.html，點右上「✎ 編輯」。
  2. 填今天的數據與焦點，選漲跌方向與判讀（偏多／偏空／中性）。
  3. 三種輸出：
     ・「匯出 brief.json」→ 覆蓋網站同資料夾的 brief.json，即完成更新。
     ・「複製 LINE 文字」→ 直接貼到 LINE 群組。
     ・「套用預覽」→ 先看效果（存在本機瀏覽器草稿）。
